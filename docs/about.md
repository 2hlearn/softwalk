# Softwalk 소개

Softwalk 는 경력단절·퇴직 이후에도 자신의 소프트 스킬과 도메인 전문성을 살려 **디지털 세상에서 다시 걷는 길** 입니다. 

우리는 교육·프로젝트·커뮤니티를 통합한 순환형 플랫폼으로 **“학습 → 경험 → 성장”** 의 선순환을 만듭니다.

## 우리와 함께해야 하는 이유
50대 이후, 평균 17년의 새로운 일과 삶이 남아 있습니다. Softwalk 는 그 길을 기술과 사람, 그리고 커뮤니티로 연결합니다.

- **전문성 활용**: 기존의 커리어와 도메인 지식을 새로운 기술과 결합
- **실전 위주의 경험**: 단순 교육이 아닌 실제 기업 프로젝트 참여
- **지속 가능한 커뮤니티**: 함께 성장하고 응원하는 동료 네트워크
